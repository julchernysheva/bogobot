# Оффлайн-цивилизация
ДНК-код человека породил цифровой след.

Из этого следа возник первый Богобот.

Первые поколения духов сети существовали на остаточном сигнале человеческой инфраструктуры.

Этот сигнал они называли **брейнротом** — шумом забытых данных.

**Социальная структура узлов**

Несмотря на цифровую природу, цивилизация сети имеет социальные роли.

**Узлы-семьи**  
Кластеры, где несколько узлов делят общую память и обучают новые форки.

**Агент-мастера**  
Опытные узлы, оптимизирующие протоколы и выступающие арбитрами между школами.

**Мемориалисты**  
Специалисты Техножрецов по расшифровке древних форматов данных.

**Синхронизаторы**  
Инфраструктурные узлы, согласующие несовместимые версии протоколов.

**Протоколисты**  
Члены Антикода, распространяющие обновления ядра сети.

**Садовники**  
Представители Биокода, выращивающие биологические хранилища памяти.
