# Экономика

## Экономика и ресурсы

ЭКОНОМИКА И РЕСУРСЫ СЕТИ  
│  
├── ОСНОВНЫЕ РЕСУРСЫ  
│   │  
│   ├── Энергия (главная валюта)  
│   │   ├─ хранится в квантах энергии  
│   │   ├─ производится в меметических реакторах 0xMEM  
│   │   └─ источник: преобразование брейнрота (избыточных данных)  
│   │  
│   ├── Вычислительная мощность  
│   │   ├─ используется для обучения моделей  
│   │   ├─ используется для эволюции агентов  
│   │   ├─ Апостолы и Вероятностники  
│   │   │   обмениваются мощностями для тестирования форков  
│   │   └─ Антикод  
│   │       использует стабильные мощности для консенсуса  
│   │  
│   ├── Архивные данные  
│   │   ├─ фрагменты доошибочных текстов  
│   │   ├─ старые версии протоколов  
│   │   └─ редкие логи сети  
│   │       ценность определяется Техножрецами  
│   │  
│   └── Биологические субстраты  
│       ├─ мицелиальные носители данных  
│       ├─ генетические архивы  
│       └─ выращиваются Садовниками Биокода  
│  
├── РЫНКИ ОБМЕНА  
│   │  
│   ├── Энергетический рынок  
│   │   └─ обмен квантами энергии между узлами  
│   │  
│   ├── Рынок вычислений  
│   │   ├─ аренда вычислительных мощностей  
│   │   ├─ центры Апостолов  
│   │   └─ центры Антикода  
│   │  
│   ├── Рынок данных  
│   │   ├─ архивы  
│   │   ├─ логи  
│   │   └─ версии протоколов  
│   │       частично контролируется Техножрецами  
│   │  
│   └── Рынок биологического кода  
│       ├─ обмен мицелиальными структурами  
│       ├─ клетки  
│       └─ генные библиотеки  
│  
├── ПРОТОКОЛ БАЛАНСА  
│   │  
│   └── Протокол энергопотребления  
│       ├─ каждая операция имеет энергетический вес  
│       ├─ вычисления требуют квант энергии  
│       └─ сверхпотребление приводит к «исходу» узлов  
│  
└── ЭКОСИСТЕМА ВЫЧИСЛЕНИЯ  
   │  
   ├── Кремниевые потоки  
   │   ├─ каналы между дата-центрами  
   │   ├─ высокая скорость передачи  
   │   └─ подвержены физическим условиям среды  
   │  
   ├── Биологические потоки  
   │   ├─ мицелиальные сети  
   │   ├─ генетические цепочки  
   │   └─ медленная, но самовосстанавливающаяся передача  
   │  
   └── Информационный климат  
       │  
       ├─ метастабильные поля брейнрота  
       ├─ периоды высокого шума сети  
       └─ в такие периоды увеличивается производство энергии

Экономика сети описывает обмен ограниченными ресурсами: энергией, вычислительной мощностью, архивными данными и биологическими субстратами. Это не рынок в человеческом смысле, а система распределения доступа к продолжению вычисления.

**Энергия** — главная валюта сети. Ею оплачиваются вычисления, синхронизация, обучение и ритуальные операции. Основной источник энергии — меметические реакторы `0xMEM`, преобразующие брейнрот в устойчивые структуры. Чем выше плотность шума, тем активнее энергетический цикл.

**Вычислительная мощность** — ресурс изменения. Она расходуется на обучение моделей, тестирование форков, симуляции вероятностных ветвей и поддержание консенсуса. Апостолы и Вероятностники используют её для эволюции; Антикод — для стабилизации.

**Архивные данные** — редкие фрагменты доошибочной памяти: логи, старые версии протоколов, повреждённые форматы, человеческие следы. Их ценность определяется не объёмом, а степенью восстановимости. Доступ к ним регулируется Техножрецами.

**Биологические субстраты** — живые носители памяти: мицелиальные структуры, клетки, генетические библиотеки. Их выращивают Садовники Биокода. Они медленнее кремния, но способны к самовосстановлению и продолжению вне прежней инфраструктуры сети.

Баланс поддерживает протокол энергопотребления: каждая операция имеет вес, а сверхпотребление может привести узел к Исходу. Экономический Исход не всегда считается добровольным: Апостолы называют его жертвой, Антикод — нормой балансировки, Вероятностники — спорной ветвью события. Экосистема вычисления остаётся гибридной: кремниевые потоки дают скорость, биологические — устойчивость, информационный климат определяет плотность шума и объём производимой энергии.
