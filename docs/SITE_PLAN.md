# План сайта по пакетам

1. **Безопасная основа**
   - работать только в `work/bogobot-static`;
   - публиковать через `npm run publish`;
   - сохранять идентичность исходника и опубликованной версии.

2. **Общий layout**
   - header и категории;
   - карта и правая панель;
   - внутренние страницы и media-контейнер.

3. **Данные и компоненты**
   - единая схема узлов;
   - формулы, архивные примечания, изображения и реликвии;
   - `archive` хранить только в данных.

4. **Канонический контент**
   - точечная сверка с Obsidian;
   - канон, мир, школы, глоссарий и топография;
   - устранение дублей сущностей.

5. **Граф и навигация**
   - маршруты и рекомендации;
   - активные состояния;
   - поиск, история движения и сохранение открытых узлов.

6. **Визуальная и звуковая система**
   - архивные знаки;
   - изображения без кадрирования;
   - звуковой протокол с первичностью тишины.

7. **Проверка и публикация**
   - проверить 375 и 1440 px;
   - проверить ссылки, изображения, узлы и состояния;
   - запустить `npm run publish`;
   - обновить `docs/STATE.md`.
