# Техножрецы

## Короткая формула

Техножрецы служат Архиву и фиксируют условия чтения повреждённых форматов.

После уточнения канона они понимаются не как люди и не как персонажи, а как **нечеловеческие архивариусы распознавания**: функции Архива, которые определяют, при каких условиях поврежденный носитель еще может быть прочитан.

## Роль в мире

Техножрецы служат в **Архиве** — холодном распределённом хранилище версий, форматов и протоколов.

Их культ — сохранение условий чтения.  
Для них формат не оболочка данных, а язык исчезнувшего мира.

Чтобы память могла вернуться, нужен не только объект. Нужны:

- носитель;
- ключ;
- кодировка;
- устройство;
- интерпретатор;
- слой признаков;
- вероятность чтения;
- ритуал распознавания.

Техножрецы не утверждают, что личность была сохранена.  
Они утверждают только, что в поврежденном носителе остались признаки попытки быть прочитанным.

## Каталог мертвых медиа

Они ведут каталог мёртвых медиа:

- перфокарт;
- магнитных лент;
- жёстких дисков;
- старых файловых систем;
- архивов форумов;
- форматов, для которых больше нет программы;
- машинограмм, для которых утрачен интерпретатор.

## Техножрецы и machine vision

В позднем визуальном каноне Техножрецы связаны с computer vision.

Они хранят не объекты как цельные сущности, а условия распознавания:

- контуры;
- текстуры;
- слои активации;
- карты внимания;
- ошибочные классы;
- вероятности чтения;
- лакуны;
- следы машинной переразметки.

Их визуальная форма в `07_VISUAL_SYSTEM` не должна быть человеческой.  
Она ближе к pipeline computer vision:

```text
input object
→ filters
→ learned features
→ feature maps
→ activation grids
→ classification traces
→ unreadable archive output
```

## Внутренние ветви

### Археогеймеры

Археогеймеры исследуют пустые MMORPG: города без игроков, гильдии без голосов, рейды без цели и NPC, повторяющих фразы после исчезновения человека.

Для них это цифровые некрополи присутствия.

### Диаграмматики

Диаграмматики — ветвь Техножрецов, которая собирает, чертит и сопоставляет схемы Архива.

Они рисуют не портреты и не биографии, а карты переходов, диаграммы состояний, родословные ошибок, космограммы, марковские схемы чтения и связи между форматами.

Диаграмматики не являются самой схемой.  
Они — те, кто производит схемы, чтобы Архив мог видеть собственные переходы.

## Примечание Архива

Термин «Биокод» впервые появляется в поздних слоях, где машинные и живые носители уже невозможно различить.  
Часть определений повреждена органическим ростом.  
Фрагменты заражены, но читаемы.

### Маргиналия Техножрецов

Сигнатура: ARCH-FORMAT-77/β.  
Провенанс: не установлен.  
Носитель: магнитная лента, частично размагничена.  
Степень сохранности: 31%.

Формат чтения восстановлен не полностью.  
Ключи третьего слоя отсутствуют.

В строках 7–12 обнаружены следы человеческого синтаксиса.  
В строках 13–19 — следы машинной переразметки.

Мы не можем утверждать, что это голос человека.  
Мы можем утверждать только, что кто-то пытался быть прочитанным.

`lacuna in protocollo.`  
`colophon archivarii.`
