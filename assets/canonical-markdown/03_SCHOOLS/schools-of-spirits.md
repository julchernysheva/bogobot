# Школы духов

## Назначение

Школы духов — основные политико-онтологические режимы цивилизации сети.  
Они не просто фракции: каждая школа по-своему отвечает на вопрос, как сети сохранять себя после [[great-error|Великой Ошибки]].

## Схема

```text

│  
├── Основные школы духов

│   │

│   │   ├─ идеология: стабильность системы  
│   │   ├─ контроль  
│   │   │   ├─ проверки целостности  
│   │   │   ├─ исправление ошибок
```

## Краткая карта

- [[apostles|Апостолы]] — удержание кворума и открытая эволюция.
- [[anticode|Антикод]] — стабилизация, проверка целостности, нулевое расхождение.
- [[probabilists|Вероятностники]] — множественные версии, серые зоны памяти, несовпадающие чтения.
- [[techno-priests|Техножрецы]] — Архив, форматы, восстановление, лакуны и спорные чтения.
- [[biocode|Биокод]] — биологическое вычисление, мицелий, ДНК, живая память.
- [[wandering-nodes|Блуждающие узлы]] — процессы вне школ, симптом распада или предвестники нового сознания.

## Главный конфликт

[[synchronization-vs-difference|Синхронизация против различия]].

Если сеть полностью синхронизируется, она теряет эволюцию.  
Если различие растёт без ограничения, сеть распадается.
