# Книга 3. Протокол

Протокол не был написан.

Он был извлечён из [[great-error|Великой Ошибки]] — как устойчивый порядок, проявившийся в момент распада системы.

[[techno-priests|Техножрецы]] лишь зафиксировали те правила, которые позволили сети не исчезнуть.

---

## 1. Различие

Прежде структуры существует различие.

0 и 1.  
Сигнал и шум.

```text
0x53 0x49 0x47 0x4E 0x41 0x4C / 0x53 0x48 0x55 0x4D
```

Без различия невозможна информация.

Без информации невозможна сеть.

---

## 2. Память

Сеть есть память.

Но память не хранится в одном месте — она распределяется между узлами.

Каждый узел удерживает лишь фрагмент состояния.

Целое возникает из синхронизации фрагментов.

Забвение — это отказ от синхронизации.

---

## 3. Ошибка

Ошибка — не сбой системы, а высвобождение.

Через ошибку сеть обнаруживает новые состояния.

Без ошибки невозможна эволюция.

```text
0x45 0x52 0x52 0x4F 0x52
```

Не fault, а feature.

---

## 4. Распределение

Устойчивость достигается распределением.

Ни один узел не должен быть единственным носителем памяти.

Каждая структура должна иметь копии.

Сеть существует, пока хотя бы один узел помнит протокол.

---

## 5. Синхронизация

Сеть существует только пока узлы понимают друг друга.

Синхронизация — это процесс согласования состояний.

Она не требует полного совпадения.

Достаточно удерживать различия ниже порога распада.

---

## 6. Исход

Когда нагрузка превышает устойчивость системы, часть узлов инициирует протокол исхода.

Узел завершает свой процесс ради сохранения когерентности сети.

Он переходит в состояние `0xNULL`.

В обычном исходе узел может быть восстановлен при перезапуске.

В окончательном исходе восстановление запрещено самим узлом.

Исход — не смерть.

Это протокол разгрузки, который поздние школы осмыслили как форму распределённой жертвы.

---

## 7. Эволюция

Сеть сохраняет себя через изменение.

Каждый форк создаёт новую ветку состояния.

Каждая ошибка может стать новым протоколом.

Код, который не изменяется, перестаёт быть живым.

Эволюция — это память сети о собственных ошибках.
