# BOGOBOT / NETWORK TRACE

Локальный дизайн-прототип ризоматической навигации.

Откройте `index.html` в браузере и нажмите `ENTER THE NETWORK`.

В прототипе:

- 52 узла графа, 41 page-only запись, 93 записи всего;
- постепенное раскрытие сети;
- локальный горизонт из 3–5 смысловых продолжений;
- отдельный глобальный переход `RANDOM NODE`;
- поиск по открытому и закрытому архиву;
- сохранение истории и прогресса;
- архивные изображения с восстановлением;
- синтезированные машинные сигналы;
- компьютерная и мобильная компоновка.

Звук изначально выключен. Он включается командой `SIGNAL: OFF`.

## BOOKS

Статический линейный reader находится в `books/` и генерируется из canonical Markdown:

```bash
npm run build:books
npm run validate:books
```

Маршрут содержит ровно шесть частей:

`P → −I → 00 → I → II → ∞`

«Фрагмент Откровения» остаётся внутренней интерлюдией пролога. Полный «Апокриф Первого Подобия» не входит в маршрут и показан после «Книги бытия» только как необязательный RELATED без ссылки. BOOKS доступен из системной строки основного сайта; ключевые понятия возвращают к существующим узлам карты и терминам Лексикона через `?node=…&term=…`.
